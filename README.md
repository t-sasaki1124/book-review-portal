# 書評ポータル

読書メモを一覧で整理・編集できる静的Webページです。

## 使い方

1. `index.html` をブラウザで開く  
2. 「新しい本を追加」から書評を作成  
3. 「編集」から内容や表紙画像、★評価を更新  
4. 左のハンドルをドラッグして並び替え  

## 機能

- 書評の追加 / 編集 / 削除
- 表紙画像のアップロード（jpg/png、縮小処理あり）
- ★評価の編集
- 検索 / ソート
- ドラッグ&ドロップで並び替え

## ファイル構成

- `index.html` : 画面レイアウト
- `styles.css` : デザイン
- `app.js` : データと動作ロジック
