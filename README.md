# 書評ポータル

読書メモを一覧で整理・編集できるWebアプリです。  
ログイン機能付き（Cognito）で、サンプル閲覧ページも用意しています。

## 起動方法

ローカルで起動する場合は、簡易サーバーを使ってください。

```bash
python -m http.server 8000
```

## URL

- ログイン: `http://localhost:8000/login.html`
- 新規登録: `http://localhost:8000/signup.html`
- サンプル閲覧（未ログイン可）: `http://localhost:8000/sample.html`
- 本番アプリ（ログイン必須）: `http://localhost:8000/index.html`

## 使い方

1. `login.html` からログイン（初回は `signup.html`）  
2. 「新しい本を追加」から書評を作成  
3. 「編集」から内容や表紙画像、★評価を更新  
4. 左のハンドルをドラッグして並び替え  

## 主な機能

- 書評の追加 / 編集 / 削除
- 表紙画像のアップロード（jpg/png、縮小処理あり）
- ★評価の編集（1〜5）
- 検索 / ソート
- ドラッグ&ドロップで並び替え
- 1行1カードの一覧表示
- Cognitoによるログイン / 新規登録 / 認証コード
- サンプル閲覧ページ（未ログインで閲覧可能）

## 操作のヒント

- **並び替え**: 左のハンドルを掴んで上下にドラッグ  
  - 「番号: 昇順」かつ検索なしのときのみ有効  
- **表紙画像**: 編集画面で`jpg/png`を選ぶと自動で縮小されます  
- **ハッシュタグ**: `#思考法 #仕事術` / `思考法, 仕事術` どちらでもOK  
- **必須項目**: タイトルのみ必須です  

## データについて

ログイン時はAPI経由でデータを保存します。  
`sample.html` は `sample.json` / `sample-data.js` を読み込む閲覧専用です。

## ファイル構成

- `index.html` : 画面レイアウト
- `styles.css` : デザイン
- `app.js` : データと動作ロジック
- `login.html` / `signup.html` : ログイン画面
- `auth.js` / `auth.css` : 認証UI
- `sample.html` : サンプル閲覧
