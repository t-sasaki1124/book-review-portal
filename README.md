# 書評ポータル

読書メモを一覧で整理・編集できる静的Webページです。  
ブラウザだけで動作し、追加・編集・並び替えができます。

## 使い方

1. `index.html` をブラウザで開く  
2. 「新しい本を追加」から書評を作成  
3. 「編集」から内容や表紙画像、★評価を更新  
4. 左のハンドルをドラッグして並び替え  

## 主な機能

- 書評の追加 / 編集 / 削除
- 表紙画像のアップロード（jpg/png、縮小処理あり）
- ★評価の編集（1〜5）
- 検索 / ソート
- ドラッグ&ドロップで並び替え
- 1行1カードの一覧表示

## 操作のヒント

- **並び替え**: 左のハンドルを掴んで上下にドラッグ  
  - 「番号: 昇順」かつ検索なしのときのみ有効  
- **表紙画像**: 編集画面で`jpg/png`を選ぶと自動で縮小されます  
- **ハッシュタグ**: `#思考法 #仕事術` / `思考法, 仕事術` どちらでもOK  
- **必須項目**: タイトルのみ必須です  

## データについて

データは`app.js`内の配列に保存されます。  
ブラウザで編集した内容は**ページを更新すると戻る**点に注意してください。  
（必要ならローカル保存機能を追加できます）

## ファイル構成

- `index.html` : 画面レイアウト
- `styles.css` : デザイン
- `app.js` : データと動作ロジック

## 追加予定の拡張例

- ローカル保存（LocalStorage）
- 画像の一括管理
- フィルタ（タグ別 / 評価別）
